# DB란?
데이터베이스는 구조화된 정보 또는 데이터의 조직화된 모음이다.<BR>
DB(데이터베이스)를 한마디로 정의하면 `데이터의 집합`이라고 할 수 있다.
<BR>

예를 들어서
```데이터 베이스에는 일상생활에서 대부분의 정보가 저장되고 관리됩니다.
카톡으로 보낸 메세지,인스타에 올린 사진 등등 이런 정보가 모두 데이터 베이스에 기록 된다.
```

데이터베이스는 일반적으로 데이터베이스 관리 시스템(DBMS)에 의해 제어된다.

SQL은 데이터 베이스에 저장된 데이터를 정의,제거,조작하기 위해 사용하는 표준 프로그래밍 언어

## DBMS이란?
`DBMS(Database Management System, 데이터베이스 관리 시스템)란?`<br> 데이터베이스를 운영하고 관리하는 소프트웨어이다

예를 들어서 
은행의 예금 계좌는 많은 사람들이 가지고 있습니다. 여러 명의 예금 계좌 정보를 모아 놓은 것이 데이터베이스입니다. 은행이 가지고 있는 예금 계좌 데이터베이스에는 여러 명이 동시에 접근할 수 있습니다. 예금 계좌 주인, 은행 직원, 인터넷 뱅킹, ATM 기기 등에서 모두 접근이 가능하니까요. 이러한 것이 가능한 이유는 바로 DBMS가 있기 때문입니다.


## SQL이란?

관계형 데이터베이스 관리 시스템(RDBMS)에서 데이터를 정의, 조작, 제어하기 위해 사용하는 표준 프로그래밍 언어

- DDL  데이터 정의어
<BR>만들기,고치기,없애기

- DCL 데이터 제어어
<BR>넣기,빼기,보기,바꾸기
- DML 데이터 조작어
<BR>허락하기,뺏기
