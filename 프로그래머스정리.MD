# 필드
자동차란 객체가 있음
자동차는 이름과 번호를 가지고 있음
이것을 `속성`이라함
자바에선 `속성`을 `필드`라고 함

이름과 번호를 필드로 가지고 있는 Car 클래스 선언
```
  public class Car{
        String name;    
        int number;
    }
```

# 메소드
객체 지향언어는 하나의 사물을 하나의 클래스로 설명하는 언어

사물에 대한 설명은 상태 행동으로 나뉨
상태->필드
행동->메소드

예
```
Car라는 클래스에서 이름과 번호가 필드,
전진과 후진이 메소드
``` 

메소드는 입력값이 있고, 그 입력값을 받아서 무언가 한 다음 결과를 도출해 내는 수학의 함수와 비슷한 개념이다.

입력값 = 매개변수<br>
결과값 = 리턴값
인자 = 어떤 함수를 호출시 전달되는 값
매개변수는 그 전달된인자를 받아들이는 변수를 의미한다

## 메소드 선언
메소드는 클래스가 가진 기능

리턴 타입이 없으면 void

### 메소드 선언 하는법
<b>- 메소드 선언하는법 - </b>
```
public 리턴타입 메소드명 (매개변수들) {구현}
```
<br>
리턴타입은 메서드 실행후 결과값 돌려주는 부분

## 매개변수도 없고 리턴하는 것도 없는 형태의 메소드
리턴하는 것이 없을 경우 `void`라고 작성합니다.
```
    public class MyClass{
        public void method1(){
            System.out.println("method1이 실행됩니다.");
        }
    }
```
<br><br>
## 정수를 받아들인 후, 리턴하지 않는 메소드
받아들이는 값은 어떤 타입이라도 상관없다.
받아 들이는 값의 수는 여러개 일 수 있다. 여러개을 경우 콤마(,)로 구분한다.
```
    public class MyClass{       
        public void method2(int x){
            System.out.println(x + " 를 이용하는 method2입니다.");
        }
    }
```
<br>

## 아무것도 받아들이지 않고, 정수를 반환하는 메소드
리턴하는 값 앞에 return 이라는 키워드를 사용한다.
메소드 이름 앞에는 리턴하는 타입을 적어준다.
리턴타입은 하나만 사용할 수 있다. 리턴하는 타입은 어떤 타입이라도 상관없다.

    public int method3(){
        System.out.println("method3이 실행됩니다.");

        return 10;
    }
    //위 메소드가 실행되면, 콘솔에 'method3이 실행됩니다.' 를 출력하고, 이 메소드를 호출한 쪽에 10을 리턴한다.
<br>

## 정수를 2개 매개변수로 받고, 아무것도 반환하지 않는 메소드

    public void method4(int x, int y){
        System.out.println(x + "," + y + " 를 이용하는 method4입니다.");
    }


＃ 메서드 사용해 보기
클래스가 가진 메소드를 사용할려면 객체를 만들어야함
